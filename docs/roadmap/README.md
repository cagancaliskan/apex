# Apex v3.0 Roadmap

This directory contains the implementation plans for the next major version of the F1 Race Strategy Workbench.

## Overview

| Phase | Name | Description | Timeline |
|-------|------|-------------|----------|
| 1 | [Intelligence Upgrade](phase1_intelligence_upgrade.md) | ML-powered pace prediction, tyre cliff detection, battle prediction | 6 weeks |
| 2 | [Visualization Upgrade](phase2_visualization_upgrade.md) | 3D WebGL track, telemetry comparison, mobile responsiveness | 5 weeks |
| 3 | [Full Race Lifecycle](phase3_full_race_lifecycle.md) | Championship simulator, multi-stop optimizer, reports, notifications | 6 weeks |

## Total Estimated Effort

**17 weeks** (can be parallelized or done incrementally)

## Getting Started

Each plan includes:
- Feature breakdowns with architecture diagrams
- Complete file listings (new/modified files)
- Backend API endpoints
- Frontend component designs
- Dependencies & verification plans
- Risk assessment & success metrics

Choose a phase to begin implementation!
